# OBS 입금 알림 Android 앱

은행 입금 문자(SMS)를 자동으로 읽어서 OBS 후원 시스템의 **실시간 입금 대기 큐**에 전송하는 앱입니다.

## 기능
- 📱 은행 입금 문자 자동 수신 및 파싱
- 💰 국민, 신한, 우리, 하나, 농협, 카카오뱅크, 토스뱅크 등 지원
- 🚀 실시간 OBS 서버 API 자동 전송
- 📋 수신 이력 로그 확인

## APK 다운로드
1. 이 레포지토리를 GitHub에 push하면 **Actions** 탭에서 자동 빌드됩니다
2. Actions → 가장 최근 빌드 → Artifacts → `OBS-Donation-APK` 다운로드

## 설치 방법
1. APK 파일을 폰에 다운로드
2. 설정 → 보안 → 알 수 없는 소스 허용
3. APK 설치
4. 앱 실행 후 SMS 권한 허용
5. 서버 주소와 프로젝트 ID 확인 후 **설정 저장**

## 기본 설정
- 서버 주소: `http://175.126.38.54/obs/api.php`
- 프로젝트 ID: `1`
